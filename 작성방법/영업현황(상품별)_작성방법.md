## 📋 이 표는 이렇게 만들어집니다

**데이터 출처**: 매월 구글 드라이브에 올라오는 CSV 4종
(공급전 / 계약전 / 시설전 / 용도변경)의 당월·전월 자료를 비교합니다.

---

### 1. 상품 그룹은 어떻게 묶이나요?

세부 상품명을 그대로 쓰면 컬럼이 너무 많아지기 때문에,
아래 표 기준으로 **5개 그룹**으로 묶어서 표시합니다.

| 묶이는 세부 상품명 | → 표시 그룹 |
|---|---|
| 취사난방용, 개별난방용, 취사용, 중앙난방용 | **가정용** |
| 일반용(1), 일반용(2) | **일반용** |
| 업무난방용, 주한미군, 냉난방용(업무), 냉난방용(주택) | **업무용** |
| 산업용 | **산업용** |
| 열병합용, 열병합용2, 자가열전용, 열전용설비용, 연료전지 | **열병합용** |

> 위 목록에 없는 상품명은 그룹으로 묶이지 않고 **원래 이름 그대로** 별도 컬럼으로 표시됩니다.
> 표에서 컬럼 순서는 `가정용 → 일반용 → 업무용 → 산업용 → 열병합용` 순으로 먼저 고정되고,
> 그 뒤에 매핑되지 않은 나머지 상품들이 **건수(절댓값) 비중이 큰 순서**로 자동으로 붙습니다.
> 즉, 매월 새로운 상품명이 생겨도 표가 자동으로 컬럼을 추가해서 보여줍니다.

---

### 2. 집계에서 제외되는 상품은 무엇인가요?

아래 상품들은 영업 실적 집계 대상이 아니라고 보고, **합계 계산 전에 아예 제거**합니다.

| 제외 조건 | 대상 |
|---|---|
| 이름이 정확히 일치 | `수송용(외주)`, `수송용(CNG)` |
| 이름이 `미사용`으로 시작 | `미사용-○○○` 형태의 모든 상품 (예: 미사용-가정용 등) |

> 위 두 조건에 해당하는 상품은 가정용/일반용 등 그룹으로 묶이기 전에 걸러지므로,
> 최종 표에는 전혀 나타나지 않습니다.

추가로, **공급전(공급 건수) 집계에서만** 적용되는 예외가 하나 더 있습니다.
- 업종이 `CES_죽곡`인 건은 건수를 0으로 처리합니다. (공급전 실적에서만 빠짐)

---

### 3. 각 항목의 의미

| 항목 | 의미 |
|---|---|
| 계약전 | 공급계약전 건수에서 폐전 건수를 뺀 순증감 |
| 시설전 | 공급계약이 끝나고 공사를 마친 뒤, 안전시설검사를 완료한 계량기 |
| 공급전 | 최초 사용계약이 완료된 계량기 |
| 폐전 | 폐전 건수 |
| 용도변경 | 상품 간 이동 (이전 상품 → 변경 상품) |

---

### 4. 핵심 지표 — 실적

> **실적 = 공급전 증감 + 폐전 증감 − 용도변경 합계**

이 보고서가 최종적으로 보여주려는 숫자이며, 표 맨 위 노란색 행에 강조 표시됩니다.

각 항목은 **당월 / 전월 / 증감** 3줄로 비교되고, 용도변경은
**유입(다른 상품에서 들어옴) / 유출(다른 상품으로 빠짐) / 합계**로 표시됩니다.
